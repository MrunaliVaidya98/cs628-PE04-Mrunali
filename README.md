# cs628-PE04-Mrunali
# Input

The user can view a list of cities and add a new city by entering the city name, country, population, and details using the Add City form. Users can also click on any city name to view its detailed information.

# Process

The React application uses React Router for navigation and routing between different pages. The Cities List page displays all cities using React state. The Add City component collects user input and updates the cities list dynamically. Nested routing and the useParams hook are used to display city details based on the selected city ID. The Navigate hook redirects users back to the Cities List page after adding a city successfully.

# Output

The application displays a user-friendly interface where users can navigate between pages, add new cities, and view city details dynamically without reloading the page.
